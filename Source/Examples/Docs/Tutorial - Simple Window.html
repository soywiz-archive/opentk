Tutorial - Simple Window

This tutorial explains how to create an OpenTK window and draw a single colored triangle.

Overview

The OpenTK.GameWindow provides the simplest method to create an OpenTK window. Your application may derive from this class and call the GameWindow constructor with the desired DisplayMode and window Title as parameters. Below is a brief overview of the GameWindow class.

Events and Methods

As soon as the GameWindow is created, you may call the GameWindow.Run method to start execution of the game loop. The game loop consists of frame Update and Render events. You can optionally set a target frame update and render frequency, which OpenTK will try to maintain.

The GameWindow.Load and Unload events occur just before entering and leaving the game loop. The OpenGL context exists and is current in both cases, making them a good place to load and dispose game resources like textures, shaders and buffers.
